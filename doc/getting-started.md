# はじめかた

## インストール

1. [Releases ページ](https://github.com/my-take-dev/myT-x/releases) から最新の `myT-x.exe` をダウンロード
2. 好きなフォルダに配置
3. ダブルクリックで起動

初回起動時に設定ファイル (`config.yaml`) が自動で作られるので、すぐ使いはじめられます。

---

## 起動後の画面

起動すると、以下のような画面が表示されます。

```
+------------------------------------------------------------+---+
|  [設定] [あ↻]  [日本語|English]                              |   |
+------------+-----------------------------------------------+ A |
|            |                                               | c |
|  myT-x     |     「セッションを作成してください」              | t |
| ────────── |                                               | i |
|            |        [▶ クイックスタート]                     | v |
|            |                                               | i |
| [+ 新規]   |                                               | t |
+------------+-----------------------------------------------+ y |
|  ステータスバー                                              |   |
+------------------------------------------------------------+---+
```

---

## 最初のセッションを作る

### 方法 1: クイックスタート（最速）

メイン画面の **▶ クイックスタート** ボタンをクリックするだけです。
デフォルトディレクトリでセッションが即座に作成されます。

### 方法 2: 新規セッション

1. 左サイドバーの **+ 新規セッション** をクリック
2. **セッション名** を入力
3. **作業ディレクトリ** を選択（「フォルダを選択...」ボタン）
4. 必要に応じてオプションを設定：
   - **Agent Team として開始** — AI チーム開発をする場合にチェック
   - **Claude Code 環境変数を使用** — 設定画面で登録した Claude Code 用の環境変数を適用
   - **ペイン環境変数を使用** — 追加の環境変数を適用
5. **作成** をクリック

> Git リポジトリ内のフォルダを選ぶと、**Worktree オプション** が自動で表示されます。
> ブランチごとに独立した作業フォルダを作れるので、複数のブランチを同時に操作できます。

## セッションの基本操作

| 操作 | やり方 |
|------|--------|
| セッション切替 | 左サイドバーのセッション名をクリック |
| セッション名変更 | セッション名をダブルクリック、または F2 キー |
| セッション並び替え | ドラッグ＆ドロップ |
| セッション終了 | 右クリック → 「セッション終了」 |
| エクスプローラーで開く | 右クリック → 「エクスプローラーで開く」 |

### セッションの種類

左サイドバーでは、セッションの種類がバッジで区別されます。

| バッジ | 種類 | 説明 |
|--------|------|------|
| **S** | 通常セッション | 標準のターミナルセッション |
| **A** | Agent Team | AI チーム開発用セッション |

Worktree を使っている場合は、リポジトリ名とブランチ名もバッジで表示されます。
