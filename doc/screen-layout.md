# 画面の見かた

## 全体レイアウト

myT-x の画面は 5 つのエリアで構成されています。

```
+------------------------------------------------------------+---+
|  ① メニューバー                                              |   |
+------------+-----------------------------------------------+ ⑤ |
|            |                                               |   |
|  ② 左      |  ③ メインエリア                                | A |
|  サイド     |                                               | c |
|  バー      |  ┌──────────┬──────────┐                      | t |
|            |  │ ペイン 1  │ ペイン 2  │                      | i |
| ────────── |  │          │          │                      | v |
|            |  └──────────┴──────────┘                      | i |
| S 作業A    |                                               | t |
| A Agent    |                                               | y |
|            |                                               |   |
| [+ 新規]   |                                               | S |
+------------+-----------------------------------------------+ t |
|  ④ ステータスバー                                            | r |
+------------------------------------------------------------+---+
```

---

## 1. メニューバー（上部）

画面上部に配置された操作バーです。

| ボタン | 説明 |
|--------|------|
| **設定** | 設定画面を開きます。シェルやキーバインド、モデル設定などを変更できます |
| **あ↻**（IME リセット） | 日本語入力が壊れた場合に押すと、入力変換を復旧します。正常時はグレーアウト |
| **日本語 / English** | 表示言語を切り替えます |

---

## 2. 左サイドバー

セッション一覧が表示されるエリアです。

### ヘッダー部分
- **myT-x** のロゴとサブタイトル
- **+ 新規セッション** ボタン

### セッションリスト

各セッションには以下の情報が表示されます：

| 表示 | 意味 |
|------|------|
| **S** / **A** バッジ | セッション種類（S = 通常、A = Agent Team） |
| セッション名 | クリックで切替、ダブルクリックで名前変更 |
| 状態バッジ | 「Selected」「Running」「Stopped」 |
| リポジトリ名 | Worktree 使用時に表示 |
| ブランチ名 | `main ↦ feature/xxx` のように表示 |

### 右クリックメニュー

セッション名を右クリックすると表示されます。

| メニュー項目 | 説明 |
|------------|------|
| ブランチに昇格 | detached HEAD をブランチ名に変換 |
| エクスプローラーで開く | 作業ディレクトリを Windows エクスプローラーで開く |
| セッション終了 | セッションを閉じる（Worktree の場合は commit/push の確認あり） |

---

## 3. メインエリア（中央）

ターミナルペインが表示されるメインの作業領域です。

### セッション未選択時
- 「セッションを作成してください」メッセージ
- **▶ クイックスタート** ボタン

### セッション選択時
ターミナルペインが配置されます。分割して複数のペインを並べられます。

各ペインには上部にツールバーがあり、以下のボタンが並びます：

| ボタン | 説明 |
|--------|------|
| **ペイン ID**（%0, %1...） | ペインの識別番号。Agent Team で他のペインを指定するときに使います |
| **ペイン名** | クリックして名前を編集できます |
| **↻ Auto Enter** | 自動入力モード。ペインが停止したとき自動でメッセージを送信します |
| **┃ 左右分割** | ペインを左右に分割します |
| **━ 上下分割** | ペインを上下に分割します |
| **👤+ メンバー追加** | Agent Team にメンバーを追加します |
| **✕ ペインを閉じる** | ペインを閉じます（赤いボタン） |

### レイアウトプリセット

複数のペインがある場合、レイアウトボタンで一括整列できます。

| プリセット | 配置 |
|-----------|------|
| 横均等 | すべてのペインを左右に均等配置 |
| 縦均等 | すべてのペインを上下に均等配置 |
| 左メイン | 左に大きなペイン、右に残りを縦に並べる |
| 上メイン | 上に大きなペイン、下に残りを横に並べる |
| タイル | グリッド状に並べる |

### キャンバスモード

通常のターミナル表示とキャンバスモードを切り替えられます。

キャンバスモードでは、各ペインがノードとして表示され、ペイン間のタスクの流れが線（エッジ）で視覚化されます。
Agent Team でオーケストレーション中の関係性がひと目でわかります。

---

## 4. ステータスバー（下部）

現在のセッション情報が表示されます。

| 表示 | 意味 |
|------|------|
| セッション名:ペイン番号 | 現在アクティブなセッションとペイン |
| ペインタイトル | 選択中のペインの名前 |
| 時刻 | 現在時刻 |
| **SYNC** | 同期入力モードが有効なときに表示 |
| **プレフィックス** | プレフィックスキーが押されたときに表示 |

---

## 5. Activity Strip（右端）

ビューアを開くためのアイコンが縦に並ぶバーです。

| アイコン | ビューア | ショートカット |
|---------|---------|-------------|
| 📁 | File Tree | Ctrl+Shift+E |
| 🌿 | Git Graph | Ctrl+Shift+G |
| ±  | Diff | Ctrl+Shift+D |
| 📜 | Input History | Ctrl+Shift+H |
| 🔌 | MCP Manager | Ctrl+Shift+M |
| ⏱ | ペインスケジューラ | Ctrl+Shift+K |
| 📋 | タスクスケジューラ | Ctrl+Shift+Q |
| 👥 | オーケストレーターチーム | Ctrl+Shift+T |
| ⚠ | Error Log | Ctrl+Shift+L |

- アイコンをクリックするとビューアが開きます
- もう一度クリック、または `Escape` キーで閉じます
- Error Log に未読がある場合、バッジ（●）が表示されます
- 下部の切替ボタンで **ドッキングモード**（横並び）と **オーバーレイモード**（重ね表示）を選べます

---

## チャット入力バー

メインエリアの下部に、ペインへのメッセージ送信バーがあります。

### 折りたたみ時
- ペイン番号と名前が表示
- クリックで展開

### 展開時（フローティングウィンドウ）

| 要素 | 説明 |
|------|------|
| ペイン選択ボタン | 送信先のペインを選びます |
| テキストエリア | メッセージを入力します（複数行可） |
| **½** ボタン | 入力欄のサイズを半分に切り替え |
| 矢印ボタン（↗↖↙↘） | 入力欄の表示位置を変更 |
| **自動で閉じる** チェック | 送信後に自動で閉じるかどうか |
| **送信** ボタン | メッセージを送信（Ctrl+Enter でも送信可） |
