# ショートカット一覧

> プレフィックスキーとビューアのショートカットは、設定画面から変更できます。

---

## グローバルショートカット

| キー | 動作 |
|------|------|
| `Ctrl+P` | クイック検索（セッション切替） |
| `Ctrl+F` | ペイン内テキスト検索 |
| `Ctrl+C` | テキスト選択中ならコピー / 未選択なら処理中断 |
| `Ctrl+V` | ペースト |
| `Ctrl` + マウスホイール | 文字サイズの拡大・縮小 |
| `Ctrl+Shift+F12` | Quake Mode（ウィンドウの表示/非表示）※設定で変更可 |
| `Escape` | 開いているビューアを閉じる |

---

## プレフィックスキー操作

デフォルトのプレフィックスキーは `Ctrl+b` です。
プレフィックスキーを押す → ステータスバーに「プレフィックス」表示 → アクションキーを押します。

| キー | 動作 |
|------|------|
| `%` | 左右に分割 |
| `"` | 上下に分割 |
| `z` | ズーム（最大化 / 元に戻す） |
| `x` | ペインを閉じる |
| `d` | デタッチ（セッションを維持して切断） |
| `s` | 同期入力モードの切替 |

---

## ビューアショートカット

| キー | ビューア |
|------|---------|
| `Ctrl+Shift+W` | Editor（エディタ） |
| `Ctrl+Shift+E` | File Tree（ファイルツリー） |
| `Ctrl+Shift+G` | Git Graph（Git グラフ） |
| `Ctrl+Shift+D` | Diff（差分ビュー） |
| `Ctrl+Shift+H` | Input History（入力履歴） |
| `Ctrl+Shift+M` | MCP Manager（MCP 管理） |
| `Ctrl+Shift+K` | Schedule（ペインスケジューラ） |
| `Ctrl+Shift+Q` | Task Queue（タスクスケジューラ） |
| `Ctrl+Shift+T` | Teams（オーケストレーターチーム） |
| `Ctrl+Shift+L` | Error Log（エラーログ） |

---

## セッション操作

| キー | 動作 |
|------|------|
| クリック | セッション切替 |
| ダブルクリック / `F2` | セッション名を変更 |
| ドラッグ＆ドロップ | セッションの並び替え |
| 右クリック | コンテキストメニューを開く |

---

## チャット入力バー

| キー | 動作 |
|------|------|
| `Ctrl+Enter` | メッセージを送信 |
| `Escape` | 入力欄を閉じる |

---

## ペインツールバー

| キー | 動作 |
|------|------|
| ペイン名をクリック | 名前を編集モードに |
| `Enter` | 名前を確定 |
| `Escape` | 名前の編集をキャンセル |

---

## ファイルツリー内

| キー | 動作 |
|------|------|
| `Ctrl+F` | ファイル検索モードを開く |
| `Enter` | 検索結果のファイルを選択 |
| `Escape` | 検索モードを閉じる |
