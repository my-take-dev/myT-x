# Keyboard Shortcuts

> The prefix key and viewer shortcuts can be customized in the settings screen.

---

## Global Shortcuts

| Key | Action |
|-----|--------|
| `Ctrl+P` | Quick search (session switcher) |
| `Ctrl+F` | Text search within pane |
| `Ctrl+C` | Copy if text selected / Interrupt if not |
| `Ctrl+V` | Paste |
| `Ctrl` + Mouse Wheel | Zoom in / out (font size) |
| `Ctrl+Shift+F12` | Quake Mode (show/hide window) — customizable |
| `Escape` | Close the active viewer |

---

## Prefix Key Actions

The default prefix key is `Ctrl+b`.
Press the prefix key → "PREFIX" appears in the status bar → Press the action key.

| Key | Action |
|-----|--------|
| `%` | Split left/right |
| `"` | Split top/bottom |
| `z` | Zoom (maximize / restore) |
| `x` | Close pane |
| `d` | Detach (keep session alive) |
| `s` | Toggle sync input mode |

---

## Viewer Shortcuts

| Key | Viewer |
|-----|--------|
| `Ctrl+Shift+E` | File Tree |
| `Ctrl+Shift+G` | Git Graph |
| `Ctrl+Shift+D` | Diff |
| `Ctrl+Shift+H` | Input History |
| `Ctrl+Shift+M` | MCP Manager |
| `Ctrl+Shift+K` | Pane Scheduler |
| `Ctrl+Shift+Q` | Task Scheduler |
| `Ctrl+Shift+T` | Orchestrator Teams |
| `Ctrl+Shift+L` | Error Log |

---

## Session Operations

| Key | Action |
|-----|--------|
| Click | Switch session |
| Double-click / `F2` | Rename session |
| Drag & Drop | Reorder sessions |
| Right-click | Open context menu |

---

## Chat Input Bar

| Key | Action |
|-----|--------|
| `Ctrl+Enter` | Send message |
| `Escape` | Close input panel |

---

## Pane Toolbar

| Key | Action |
|-----|--------|
| Click pane name | Enter edit mode |
| `Enter` | Confirm name |
| `Escape` | Cancel editing |

---

## Inside File Tree

| Key | Action |
|-----|--------|
| `Ctrl+F` | Open file search |
| `Enter` | Select file from search results |
| `Escape` | Close search |
