# ターミナル操作

## 画面分割

### ツールバーから

各ペインの上部ツールバーにあるボタンで分割できます。

| ボタン | 操作 | 結果 |
|--------|------|------|
| ┃ | 左右分割 | 現在のペインの右に新しいペインが生まれます |
| ━ | 上下分割 | 現在のペインの下に新しいペインが生まれます |

### プレフィックスキーから

1. `Ctrl+b`（プレフィックスキー）を押す → ステータスバーに「プレフィックス」と表示
2. 続けてキーを押す

| キー | 操作 |
|------|------|
| `%` | 左右分割 |
| `"` | 上下分割 |
| `z` | ズーム（最大化 / 元に戻す） |
| `x` | ペインを閉じる |
| `d` | デタッチ（セッションを維持して切断） |
| `s` | 同期入力モードの切替 |

> プレフィックスキーとその後のキーは、設定画面から変更できます。

---

## レイアウト整列

複数ペインがあるとき、レイアウトボタンで一発で整列できます。

| プリセット | イメージ |
|-----------|---------|
| **横均等** | `[A][B][C]` — 全ペインを左右に並べる |
| **縦均等** | `[A]` / `[B]` / `[C]` — 全ペインを上下に並べる |
| **左メイン** | `[A  ][B]` / `[A  ][C]` — 左に大きなペイン |
| **上メイン** | `[  A  ]` / `[B][C]` — 上に大きなペイン |
| **タイル** | `[A][B]` / `[C][D]` — グリッド配置 |

---

## コピー＆ペースト

| 操作 | やり方 |
|------|--------|
| コピー | テキストを選択するだけで自動コピー。`Ctrl+C`（テキスト選択中）でもOK |
| ペースト | 右クリック、または `Ctrl+V` |
| 処理中断 | `Ctrl+C`（テキスト未選択時） |

---

## 文字サイズ変更

`Ctrl` + マウスホイールで拡大・縮小できます。

---

## テキスト検索

`Ctrl+F` でペイン内のテキストを検索できます。
長いログ出力の中からエラーメッセージを探すときに便利です。

---

## ファイルドロップ

ファイルをペインにドラッグ＆ドロップすると、そのファイルのパスが入力されます。

---

## クイック検索

`Ctrl+P` でクイック検索パレットが開きます。

- セッション名、リポジトリ名、ブランチ名であいまい検索できます
- 矢印キーで候補を選び、`Enter` で切り替え
- `Escape` で閉じます

たくさんのセッションがある場合に、素早く目的のセッションに移動できます。

---

## Quake Mode（すぐ呼び出し）

`Ctrl+Shift+F12`（設定で変更可）を押すと、ウィンドウが表示 / 非表示になります。

ブラウザやエディタで作業しているときに、ターミナルが必要な瞬間だけ呼び出す使い方ができます。

> 設定画面の「基本設定」タブから ON/OFF の切替とホットキーの変更ができます。

---

## 同期入力モード

同期入力モードを有効にすると、入力内容がウィンドウ内の **すべてのペイン** に同時送信されます。

### 有効にする方法
- プレフィックスキー → `s` を押す
- またはセッションビュー上部のトグルボタン

### 使い道
- 複数のペインで同じコマンドを一斉に実行したいとき
- Agent Team の全エージェントに同じ指示を送りたいとき

ステータスバーに **SYNC** と表示されているときは同期入力モードが有効です。

---

## チャット入力バー

ペインに直接メッセージを送信できるフローティング入力欄です。

### 基本の使い方

1. 画面下部の入力バーをクリックして展開
2. 送信先のペインをボタンで選択
3. メッセージを入力
4. **送信** ボタン、または `Ctrl+Enter` で送信

### オプション

| オプション | 説明 |
|-----------|------|
| **½** ボタン | 入力欄のサイズを半分に縮小/復元 |
| 矢印ボタン（↗↖↙↘） | 入力欄の表示位置（右上/左上/左下/右下）を変更 |
| **自動で閉じる** | チェックONで、送信後に入力欄が自動で閉じます |

---

## Auto Enter（自動入力）

ペインのツールバーにある **↻ Auto Enter** ボタンで設定します。

有効にすると、ペインのプロセスが停止（アイドル）になったとき、設定したメッセージが自動で送信されます。
CI/CD のような繰り返し作業や、AI エージェントへの自動指示に便利です。

---

## ペイン名の編集

ツールバーのペイン名部分をクリックすると編集モードになります。

- `Enter` で確定
- `Escape` でキャンセル

Agent Team でのオーケストレーション時に、各ペインの役割がわかりやすくなります。
