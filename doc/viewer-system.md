# ビューアシステム

画面右端の **Activity Strip** からアクセスする 9 つの開発ツールです。
アイコンをクリック、またはキーボードショートカットで開閉できます。`Escape` で閉じます。

## 表示モード

Activity Strip 下部のボタンで切り替えられます。

| モード | 説明 |
|--------|------|
| **オーバーレイ** | メインエリアの上に重ねて表示。ターミナルが隠れます |
| **ドッキング** | メインエリアの右に横並びで表示。境界線をドラッグして幅を調整できます |

---

## 1. File Tree（ファイルツリー）

**ショートカット:** `Ctrl+Shift+E`

セッションの作業ディレクトリ内のファイルとフォルダをツリー表示します。

### 左パネル（ツリー / 検索）

| 操作 | やり方 |
|------|--------|
| フォルダを開く/閉じる | フォルダアイコンをクリック |
| ファイルを選択 | ファイル名をクリック → 右パネルにプレビュー表示 |
| ファイル検索 | `Ctrl+F` で検索モードに切替。名前やパスで検索 |

### 右パネル（ファイルプレビュー）

| ファイル種類 | 表示 |
|------------|------|
| テキスト / コード | シンタックスハイライト付きで表示 |
| Markdown | プレビュー表示 |
| バイナリ | 「(Binary file)」と表示 |

### ヘッダーボタン

| ボタン | 説明 |
|--------|------|
| ↻ | ファイルツリーを再読込 |
| ✕ | ビューアを閉じる |

---

## 2. Git Graph（Git グラフ）

**ショートカット:** `Ctrl+Shift+G`

コミット履歴を SVG グラフで視覚的に表示します。

### ヘッダー

| 要素 | 説明 |
|------|------|
| **All Branches** チェック | ON にすると全ブランチ（リモート含む）を表示 |
| ↻ | 再読込 |
| ✕ | 閉じる |

### ブランチ状態バー

現在のブランチ名とトラッキング情報を表示します。

### 左パネル（コミットグラフ）

- ブランチの分岐・マージが線で表示されます
- コミットをクリックすると右パネルに詳細を表示
- 「Load more...」で過去のコミットを追加読み込み

### 右パネル（コミット詳細）

コミットを選択すると表示されます。

| 表示 | 内容 |
|------|------|
| コミットハッシュ | クリックでコピー |
| 作者名 | コミットの作成者 |
| コミットメッセージ | コミットの内容 |
| Diff | 変更されたファイルと行単位の差分 |

## 3. Diff（差分ビュー）

**ショートカット:** `Ctrl+Shift+D`

`git diff HEAD` の結果をビジュアルに表示します。ステージング操作もここから行えます。

### ヘッダー

| 要素 | 説明 |
|------|------|
| **ツリー / フラット** 切替 | 表示モードを変更 |
| ↻ | 再読込 |
| ✕ | 閉じる |

### 統計バー

変更の概要が表示されます：`+追加行数` `-削除行数` `Files: ファイル数`

### ツリーモード

**左パネル（ファイルツリー）**
- ディレクトリ構造で変更ファイルを表示
- ファイルをクリックすると差分を表示

**右パネル（コミットパネル）**
- ブランチ情報
- **コミットメッセージ** テキストエリア
- Git操作ボタン：

| ボタン | 説明 |
|--------|------|
| **Pull** | リモートから最新を取得してマージ |
| **Fetch** | リモートから最新を取得（マージなし） |
| **Commit** | ステージ済みの変更をコミット |
| **Commit & Push** | コミットしてリモートに送信 |
| **Push** | コミット済みの変更をリモートに送信 |

### フラットモード（ステージングリスト）

すべての変更ファイルを一覧表示し、ステージング操作ができます。

| 操作 | やり方 |
|------|--------|
| ファイルをステージ | ファイル行の＋ボタン |
| ファイルをアンステージ | ファイル行の－ボタン |
| 変更を破棄 | ファイル行の破棄ボタン（確認あり） |
| 全てステージ | **Stage All** ボタン |
| 全てアンステージ | **Unstage All** ボタン |

### 差分表示

| 表示 | 意味 |
|------|------|
| 緑背景 | 追加された行 |
| 赤背景 | 削除された行 |
| 灰色背景 | 変更なしの文脈行 |
| 「Expand hidden lines」 | 折りたたまれた行を展開 |

## 4. Input History（入力履歴）

**ショートカット:** `Ctrl+Shift+H`

ペインに送信されたコマンド・メッセージの履歴を表示します。

| 表示 | 内容 |
|------|------|
| タイムスタンプ | 入力した時刻 |
| ペイン ID | どのペインで実行したか |
| 入力テキスト | 送信した内容 |
| ソース | 入力元（キーボード、チャット、同期入力） |

### ヘッダーボタン

| ボタン | 説明 |
|--------|------|
| コピー | すべてのエントリをクリップボードにコピー |
| 既読にする | すべての未読バッジをクリア |
| ↻ | 再読込 |
| ✕ | 閉じる |

---

## 5. MCP Manager（MCP 管理）

**ショートカット:** `Ctrl+Shift+M`

セッションで利用可能な MCP（Model Context Protocol）サーバーを管理します。

### カテゴリ

| カテゴリ | 内容 |
|---------|------|
| **Agent Orchestrator** | オーケストレーション用 MCP。エージェント間通信に使用 |
| **LSP-MCP** | 言語サーバー MCP。200 種以上の言語拡張をサポート |

### 操作

- カテゴリをクリックして切替
- 各 MCP の接続情報を確認
- MCP の有効/無効を切替

### 外部接続

MCP Manager には、外部の AI ツール（Claude Code、Codex CLI、Gemini CLI）から接続するための情報が表示されます。
`--session` フラグの使い方もここで確認できます。

## 6. ペインスケジューラ（Schedule）

**ショートカット:** `Ctrl+Shift+K`

ペインに対して定期的にメッセージを送信するスケジューラです。

### スケジューラリスト画面

| 表示 | 内容 |
|------|------|
| スケジューラ名 | タスクの名前 |
| 状態 | 実行中 / 停止 / 完了 |
| 実行回数 | これまでの実行回数 |
| 次の実行 | 次に送信されるまでの時間 |

| ボタン | 説明 |
|--------|------|
| ✏ 編集 | スケジューラの設定を変更 |
| 🗑 削除 | スケジューラを削除 |
| ▶ / ⏸ | 開始 / 停止 |
| **+ 新規** | 新しいスケジューラを作成 |

### スケジューラ設定画面

| 項目 | 説明 |
|------|------|
| タイトル | スケジューラの名前 |
| 対象ペイン | メッセージを送信するペイン |
| メッセージ | 送信する内容 |
| 間隔（秒） | 送信間隔 |
| 最大回数 | 0 で無制限 |

---

## 7. タスクスケジューラ（Task Queue）

**ショートカット:** `Ctrl+Shift+Q`

タスクをキューに登録し、順番に自動実行するスケジューラです。

### タスクキュー画面

| 表示 | 内容 |
|------|------|
| タスク名 | タスクの名前 |
| 状態 | pending / completed / failed / abandoned |
| 実行時間 | タスクの所要時間 |

| ボタン | 説明 |
|--------|------|
| **+ 新規タスク** | タスクを追加 |
| ✏ 編集 | タスクの内容を変更 |
| 🗑 削除 | タスクをキューから除去 |
| ▶ キュー開始 | 登録済みタスクの順次実行を開始 |
| ⏸ 一時停止 | キューの実行を一時停止 |
| ⏹ 停止 | キューを完全停止（残りは skipped） |

### タスク登録画面

| 項目 | 説明 |
|------|------|
| タイトル | タスクの名前 |
| メッセージ | ペインに送信するコマンドや指示 |
| 対象ペイン | 送信先のペイン |
| **クリアしてから実行** | 実行前にペインをクリアするか |
| クリアコマンド | クリアに使うコマンド（例: `clear`） |

### テンプレート

よく使うタスクの組み合わせをテンプレートとして保存・呼び出しできます。

## 9. Error Log（エラーログ）

**ショートカット:** `Ctrl+Shift+L`

アプリケーションの Warn / Error レベルのログをリアルタイムで表示します。

| 表示 | 内容 |
|------|------|
| タイムスタンプ | エラー発生時刻 |
| レベル | error / warn |
| メッセージ | エラーの内容 |
| ソース | エラーが発生したコンポーネント名 |

### ヘッダーボタン

| ボタン | 説明 |
|--------|------|
| コピー | すべてのエントリをクリップボードにコピー |
| 既読にする | 未読バッジをクリア |
| ↻ | 再読込 |
| ✕ | 閉じる |

### バッジ通知

未読のエラーがある場合、Activity Strip の Error Log アイコンにバッジ（●）が表示されます。

ログは JSONL 形式でファイルに保存されるため、セッション終了後も確認できます。
